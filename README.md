- Spotify Premium

## Tech Stack

### Frontend

- Next.js 14
- TypeScript
- Axios

### Backend

- Node.js
- Express.js
- MySQL
- Sequelize

---

## Installation

### Backend

```bash
cd backend

npm install

cp .env.example .env

npm run dev
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```

---

## Build Frontend

```bash
npm run build
npm start
```

---

## Deployment

### Frontend (Vercel)

```bash
vercel
```

### Backend (Render)

```bash
git push github main
```

Set Environment Variables:

- DB_HOST
- DB_NAME
- DB_USER
- DB_PASSWORD
- JWT_SECRET
