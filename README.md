Premium Subscription Selling Platform

- Veo Plans
- YouTube Premium
- Spotify Premium

- bKash
- Nagad
- Rocket
- Upay
- Cellfin

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Admin Panel

```bash
cd admin-panel
npm install
npm run dev
```
